# butterfly_gesture_magic
🦋 3D AR 魔法指尖 2.1：MediaPipe & Three.js 蝴蝶飞舞特技

这是一个基于 Web 技术构建的高性能、沉浸式 3D AR（增强现实）手势互动特效网页。项目通过底层全屏调用前置/后置摄像头，将透明的 3D 渲染层完美重合在相机之上，利用 MediaPipe Hands 实时追踪手部骨骼及姿态，并在 3D 空间中的指尖与掌心诞生优雅飞舞的 3D 霓虹蝴蝶与漫天星砂粒子。

本项目的视觉交互布局与 AR 贴合逻辑参考了经典交互效果，旨在创造一种自然、梦幻的镜面魔法体验。

✨ 2.1 物理引擎升级核心特性

✊ 360° 超级烟花大爆炸 (Fist Burst)：【重磅物理升级】 握拳蓄力释放后，产生真正的球体全向各向同性超级大爆炸效果！蝴蝶爆破数量呈几何级数疯狂暴增至 150 - 350 只，同时配合 300 - 500 个高密度霓虹粒子冲击波。引入双层同心爆裂算法（外圈极速推进，内壳温和扩散）与动态三维强光瞬时闪烁（Ambient Flash），营造出极其震撼、宛若实体烟花般璀璨的顶级全向爆裂效果。

👌 指尖捏合栖息 (Pinch & Perch)：大拇指与食指捏合时，一只精致的 3D 蝴蝶会瞬间诞生，并且温顺地停留在你的指尖上轻轻扇翅膀。栖息期间它会一直陪伴着你且不消耗生命，松开手指时蝴蝶会被赋予一个向上的物理冲力翩翩飞去。

🌀 快速划动星砂 (Swirl & Dust)：当手快速划动时，运动轨迹会自动爆发出不同尺寸（巨型、常规、迷你型）的蝴蝶群落，并拖曳出多色交错、具有厚重感的浓密彩色霓虹星尘粒子长河（视觉效果向参考图完美看齐）。

真·AR 视口融合：将摄像头流作为全屏背景，将 Three.js 画布设置为透明并绝对定位叠加在顶层，实现“蝴蝶在手尖飞出”的增强现实观感。

慢速空灵物理运动：针对视觉舒适度进行了物理算法优化，减缓了蝴蝶日常升空与振翅频率，使其在空中滑过优雅的正弦曲线，更具仙气与仪式感。

丰富的控制面板：

四种魔法色系切换：幻彩（紫色/粉色/绿）、极光（红/深紫/幽蓝）、金澄（橘/黄/烈焰红）、幽蓝（冰霜蓝/淡天蓝/深海蓝）。

骨骼连线开关：一键切换显示/隐藏 MediaPipe 的 21 点手部紫金色骨骼追踪物理连线。

镜面/前后摄像头翻转：支持移动端后置摄像头，并自适应对 3D 投影和 2D 骨骼进行镜像对称翻转。

实时帧率监测：包含精简的 FPS 指示器，便于进行性能调试与调优。

🛠️ 技术栈与依赖

该项目采用纯正的 Vanilla JS、HTML 和 CSS 构建，采用免构建的 Single-file（单文件）结构，所有依赖均采用极速 CDN 引入：

渲染引擎：Three.js (r128) —— 负责蝴蝶的 3D 建模、光影特效、粒子动画及 3D 空间坐标系变换。

AI 手势识别：Google MediaPipe Hands —— 负责单手 21 个骨骼关键点（Landmarks）的高帧率三维空间追踪。

样式框架：Tailwind CSS —— 用于响应式、毛玻璃风格的魔幻控制层 UI 布局。

图标库：FontAwesome 6.4 —— 提供精致的控制栏图标。

⚙️ 常见问题解答与排查 (Troubleshooting)

页面打开后一片漆黑，提示“相机授权失败”？

原因：浏览器没有获得摄像头使用权限，或者未处于 https:// 的安全域名下。

解决方法：请检查浏览器地址栏左侧是否有红色叹号或锁头图标，点击并手动允许摄像头权限；请确保使用的是安全协议（本地调试用 localhost，线上分享必须用带 https 的链接）。

蝴蝶特效跟我的手是反的？

解决方法：点击右上角控制面板中的 “翻转镜头” 按钮。程序会自动在 user（前置镜面模式）和 environment（后置直视模式）之间切换，并实时翻转 Three.js 视窗与骨骼定位，确保运动反馈与物理直觉一致。

画面卡顿或延迟较高？

优化建议：MediaPipe 对设备的 CPU 算力有一定要求。如果运行不畅，可以在右下角点击 “骨骼显示：开” 切换至 “关”，关闭 2D 骨骼的实时 Canvas 绘制，这样可以大幅减轻双重渲染的负担，提升 3D 特效运行帧率。
